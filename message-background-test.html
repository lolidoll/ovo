💻 手动命令

在浏览器控制台中执行以下命令：

打开诊断工具

window.MessageBackgroundDiagnostics.diagnose()

直接打开管理器

window.MessageBackgroundManagerUI?.open()

查看所有背景

window.MessageBackgroundManager?.getBackgrounds()

清除所有背景

window.MessageBackgroundManager?.clearBackground()

获取当前背景 ID

window.MessageBackgroundManager?.currentBackgroundId

🔧 故障排除

如果点击"消息背景"没有反应，请按照以下步骤：
  1. 按 F12 打开开发者工具
  2. 切换到 Console 标签
  3. 点击上面的"运行诊断"按钮
  4. 查看输出结果
  5. 根据结果参考 MESSAGE_BACKGROUND_TROUBLESHOOTING.md

常见解决方案

  • 清除浏览器缓存：按 Ctrl+Shift+Delete
  • 硬刷新：按 Ctrl+F5
  • 重启浏览器：完全关闭后重新打开
